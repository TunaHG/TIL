    *   메소드 참조로 사용하니까 메소드를 인수로 사용하는 곳에서 사용하는 것 같은데...?
    *   위에 코드는 메소드를 인수로 쓰는게 아니라 그냥 메소드를 쓰는거라 적용하는 대상이 아닌것 같고
    *   아니다. 메소드를 인수로 쓴다는게 그게 아니지.

        *   이 경우 charAt()의 인수로 메소드가 들어가 있는데 결국은 메소드의 반환값을 사용하는 거니까...!
    *   정적 메소드는 static 메소드이다!
